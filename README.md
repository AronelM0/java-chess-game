# Java Chess Game

Chess game developed in Java for the OOP course.

## Author
Sianu Aronel Mihaita  


## Project overview
This application implements a playable chess game with a graphical interface built using Swing.

## Main features
- chess rules validation
- check and checkmate detection
- login and menu flow
- save/load functionality
- local match history
- graphical user interface

## Design patterns / architecture
- Observer
- Singleton
- separation between game logic and GUI

## Main challenges
- implementing correct chess move validation
- handling check and checkmate detection
- building a responsive Swing interface
- implementing save/load through serialization
